# PlainSight Lab — Public Brand Guide
## Visual Identity Only (Public)

**Status:** Public-facing visual standards  
**Scope:** Visual identity, layout, and presentation only  
**Explicitly Excluded:** Voice, tone, mission, governance language, authority framing

This document defines the **visual constraints** for PlainSight Lab public properties, including websites, documentation portals, and open repositories. It intentionally omits voice, tone, and philosophical guidance.

---

## Color Palette

### Primary Colors

**Charcoal — Authority**
- Hex: `#1A1A1A`
- Usage: Primary text, headers, logos, institutional weight elements

**Off-White — Clarity**
- Hex: `#E8E6E1`
- Usage: Primary background, content surfaces, whitespace

**Slate — Structure**
- Hex: `#4A5568`
- Usage: Secondary text, borders, dividers

### Accent Colors

**Steel Blue — Emphasis**
- Hex: `#2C5F7E`
- Usage: Links, interactive elements, focus states

**Crimson — Warning**
- Hex: `#8B1E1E`
- Usage: Alerts, boundary conditions (use sparingly)

**Frost — Precision**
- Hex: `#C1D5E0`
- Usage: Code blocks, data tables, technical surfaces

---

## Typography

### Primary Typeface

**Inter (sans-serif)**  
Used for all body text, headings, navigation, and UI labels.

**Font Weights**
- 400 — Body text
- 500 — Emphasis / subheadings
- 600 — Section headers
- 700 — Primary headers

Fallback stack:
```
-apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, sans-serif
```

### Monospace Typeface

**JetBrains Mono**  
Used for:
- Code blocks
- Technical specifications
- Data output
- Inline code references

## Spacing & Layout

### Core Principles
- High whitespace density
- Predictable spacing
- Explicit section boundaries
- One concept per visual block

### Spacing Scale

| Token | Value |
|------|------|
| xs | 4px |
| sm | 8px |
| md | 16px |
| lg | 24px |
| xl | 32px |
| 2xl | 48px |
| 3xl | 64px |

### Layout Constraints
- Max content width: `1264px`
- Default container: `min(1264px, 92vw)`
- Content centered with generous margins
- Grid-based layouts preferred over freeform

---

## UI Elements

### Buttons
- Border radius: 4px
- No pill shapes
- Minimal shadow only if needed for separation

### Borders & Dividers
- Use Slate (`#4A5568`) or Frost (`#C1D5E0`)
- 1px–2px thickness
- No decorative lines

### Code Blocks
- Background: Charcoal (`#1A1A1A`)
- Text: Off-White (`#E8E6E1`)
- Padding: 16px
- Border radius: 4px

---

## Prohibited Visual Patterns

- Gradients (except subtle, structural use)
- Rounded pill buttons
- Decorative illustrations
- Emoji
- Stock photography with people
- High-saturation colors

---

## Binding Statement

This public brand guide defines **mandatory visual constraints** for PlainSight Lab public-facing properties.

Deviation requires explicit approval by PlainSight Lab governance.
