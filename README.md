# PlainSightLab.com Site Scaffold

This repository contains the Hugo site scaffold for PlainSightLabs.com.

## Local Development

1. Ensure Hugo is installed.
2. Initialize the theme submodule:
   - `git submodule update --init --recursive`
3. Run the server:
   - `hugo server`

## Theme

The site uses the `hugo-theme-aperture` theme as a git submodule at `themes/hugo-theme-aperture`.
